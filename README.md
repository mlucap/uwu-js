# uwu-js 
[![npm version](https://badge.fury.io/js/uwu-js.svg)](https://badge.fury.io/js/uwu-js)

Helps make your text uwu-ified

## Installation
`npm i uwu-js --save`

## Usage
```
const uwu = require('uwu-js');
console.log(uwu('Hello There!')); // Hewwo Thewe (｡♥‿♥｡)
```
